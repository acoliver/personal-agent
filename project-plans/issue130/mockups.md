### Download Flow

```
                    +----------------+
                    | Start Download |
                    +-------+--------+
                            |
                            v
                    +-------+--------+
                    | Create HfClient |
                    +-------+--------+
                            |
                            v
            +---------------+---------------+
            | HfClient.download_file()     |
            | - Uses HTTP range requests   |
            | - Resumes if partial exists  |
            | - Reports progress           |
            +---------------+---------------+
                            |
                            v
                +-----------+-----------+
                | Download succeeded?   |
                +-----------+-----------+
                            |
               +------------+------------+
               |                         |
               v                         v
        +------+------+           +------+------+
        |     Yes      |           |     No      |
        +------+------+           +------+------+
               |                         |
               v                         v
        +------+------+    +-------------+-------------+
        | Verify SHA256  |    | Show error modal       |
        | checksum       |    | with retry option      |
        +------+------+    +-------------------------+
               |
               v
    +----------+----------+
    | Verification OK?    |
    +----------+----------+
               |
          +----+----+
          |         |
          v         v
       +--+--+  +--+--+
       | Yes |  | No  |
       +--+--+  +--+--+
          |         |
          v         v
   +------+------+  +-------------+-------------+
   | Create profile |  Delete corrupt, retry   |
   | "Local AI"     |  download                 |
   +------+------+  +--------------------------+
          |
          v
   +------+------+
   | Notify user |
   | "Ready!"    |
   +-------------+
```
