Error History View - accessed by clicking the bug icon.

- ← Back returns to Chat view (stack navigation).
- Errors in reverse chronological order (newest first).
- Each entry: severity tag (Stream/Auth/Conn/MCP), source (provider/model), timestamp, error message, conversation context.
- Expandable raw response preserves full error detail.
- Clear All empties the ring buffer.
- Viewing this list clears the unviewed count on the bug icon.
- Ring buffer: last ~100 errors in memory (no disk persistence).
- All colors via Theme::* tokens - adapts to any theme.
