## Audit Scope Domain Review

### Theme Assets Copy Ownership and Loader/Schema

- **16 bundled theme JSON files** in `assets/themes/` (ansi, ansi-light, atom-one-dark, ayu, ayu-light, default, default-light, dracula, github-dark, github-light, googlecode, green-screen, no-color, shades-of-purple, xcode).
- Theme catalog loaded at runtime via `available_themes()` in `src/ui_gpui/theme.rs`.
- JSON schema defines color tokens: `bg`, `fg`, `accent`, `border`, `input_bg`, `input_fg`, `input_border`, `button_bg`, `button_fg`, `button_border`, `error`, `success`, `warning`, `scrollbar`, `shadow`.
- `set_active_theme_slug()` / `active_theme_slug()` manage global theme state.

**Status: [OK] Verified**

### Runtime Theme Engine File-Driven Behavior

- `resolve_active_palette()` loads the active theme from catalog JSON files at runtime.
- Fallback palette (`FALLBACK_PALETTE`) provides zero-catalog resilience.
- `ThemePalette` struct resolves all color tokens via `resolve()` and `resolve_with_mac_native()`.
- 45 theme-related tests cover color resolution, helper functions, and palette behavior.

**Status: [OK] Verified**

### Settings Theme Dropdown/Event/Persistence Wiring

- Settings view in `src/ui_gpui/views/settings_view/` contains theme dropdown rendering (`render.rs`), command handling (`command.rs`), and state management (`mod.rs`).
- `ThemeOption` struct provides `name` and `slug` for dropdown population.
- `select_theme` event persists via `SettingsPresenter` → `AppSettingsService` → disk.
- Test `theme_dropdown_options_select_expected_entry_and_fallback` in `tests/settings_view_tests.rs` validates dropdown behavior.
- `settings_presenter::select_theme_persists_and_re_emits_theme_snapshot` confirms persistence round-trip.

**Status: [OK] Verified**

### Mac-Native Option Semantics and Fallback

- `mac_native` module (`src/ui_gpui/mac_native.rs`) provides `MacNativePalette` and `resolve_palette()`.
- `MAC_NATIVE_SLUG` constant used throughout.
- `try_mac_native_palette()` safely returns `None` when AppKit is unavailable.
- `resolve_with_mac_native()` checks active slug and falls back to catalog palette.
- Synthetic `mac-native` pseudo-entry appended to theme options list.
- macOS-conditional compilation via `#[cfg(target_os = "macos")]`.

**Status: [OK] Verified**

### Legacy Migration Mapping dark/light/auto

- `migrate_legacy_theme_slug()` in `src/ui_gpui/theme.rs` maps:
  - `"dark"` → `"default"`
  - `"light"` → `"default-light"`
  - `"auto"` → `"mac-native"`
- Applied at startup in `src/main_gpui/startup.rs` before first render.
- Unrecognized slugs pass through unchanged.

**Status: [OK] Verified**
