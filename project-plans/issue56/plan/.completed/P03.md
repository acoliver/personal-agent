### 1. `src/services/conversation_sqlite.rs` — Created

Full `ConversationService` trait implementation backed by SQLite via `DbHandle`.

**Struct:**
```rust
pub struct SqliteConversationService {
    db: DbHandle,
    active_id: Mutex<Option<Uuid>>,
}
```

**All trait methods implemented:**

| Method | Implementation |
|--------|----------------|
| `create` | INSERT into conversations with UUID, RFC3339 timestamps |
| `load` | SELECT conversation row + all messages ORDER BY seq |
| `list_metadata` | SQL with correlated subqueries per spec §6.0 |
| `add_message` | Transaction: INSERT (seq via COALESCE MAX+1) + UPDATE updated_at, per spec §5.1 |
| `search` | FTS5 CTE query per spec §4.1; sanitization per spec §4.2; post-query match type classification in Rust |
| `message_count` | SELECT COUNT(*) |
| `rename` | UPDATE title + updated_at; triggers handle FTS sync |
| `delete` | DELETE cascade; triggers handle FTS cleanup |
| `set_active` | SELECT 1 verify exists, then update Mutex |
| `get_active` | Read from Mutex |
| `get_messages` | SELECT ORDER BY seq ASC |
| `update` | COALESCE UPDATE for optional fields, then load |
| `update_context_state` | Serialize ContextState to JSON, UPDATE; bumps updated_at |
| `get_context_state` | SELECT context_state, deserialize JSON; None on NULL or parse failure (logged at warn) |

**Key implementation details:**
- Timestamps: `chrono::Utc::now().to_rfc3339_opts(SecondsFormat::Millis, true)` → Z suffix
- Seq allocation: `COALESCE((SELECT MAX(seq) + 1 FROM messages WHERE conversation_id = ?), 0)` in transaction
- FTS5 sanitization: strip `* ( ) { } ^ ~ :` with spaces, quote each term, prefix wildcard on last term
- Error mapping: `QueryReturnedNoRows` → `NotFound`, constraint violations → `NotFound`, all others → `Storage`
- `add_message`: stores caller-provided `message.timestamp` (does not override with Utc::now())
- `update_context_state`: bumps `updated_at` per spec §5.6
- `get_context_state`: deserialization failures logged at `warn` and returned as `None`
