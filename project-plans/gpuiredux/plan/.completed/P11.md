## Navigation Flow

1. MainPanel starts with NavigationState at ViewId::Chat
2. UserEvent::NavigateBack or navigate to Settings/History/etc triggers ViewCommand::NavigateTo
3. MainPanel.handle_command() calls navigation.navigate(view)
4. render() checks navigation.current() and renders the appropriate view entity
5. Each view's Cancel button emits UserEvent::NavigateBack → ViewCommand::NavigateBack → navigation.navigate_back()
