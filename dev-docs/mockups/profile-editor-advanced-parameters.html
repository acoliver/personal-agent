5. Direction Summary
Collapsed state: keep Advanced request parameters as a simple disclosure row.
Advanced mapping: let users change MAX TOKENS FIELD NAME from the default max_tokens to max_completion_tokens or another compatible name.
Omission behavior: if the user clears the main MAX TOKENS value, the request sends no token-limit field at all.
Extra JSON: still available for provider-specific fields that do not fit the typed editor.
