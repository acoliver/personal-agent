5. Implementation Intent
Proposed IA: add a top-level Skills category between Security and MCP Tools.
Skills panel owns: discovered skills list, watched directories, URL import, refresh, reveal/open actions, install target selection, and empty-state education.
Security keeps: only Auto-approve skill activation next to the other approval controls.
Expected UX improvement: no more invisible system. A user can immediately see where skills come from, where they are stored, and how to add one without guessing.
